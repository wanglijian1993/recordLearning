# Linux学习路线

### **1.熟练运用linux命令行推荐书籍**

《**鸟哥的Linux私房菜**》。如果想再深入一点，推荐你阅读《**Linux系统管理技术手册**》。

### **2.通过系统调用或者glibc，学会自己进行程序设计**

推荐你阅读**《UNIX环境高级编程》**

### **3.了解Linux内核机制，反复研习重点突破**

推荐你阅读**深入理解LINUX内核**

### **第四坡：阅读Linux内核代码，聚焦核心逻辑和场景**

推荐书籍《**LINUX内核源代码情景分析**》
